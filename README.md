## Documentation

### Initializing

``` python
from tapioca_twitter import Twitter

api = Twitter(
    api_key='{your-api-id}',
    api_secret='{your-api-secret}',
    access_token='{your-access-token}',
    access_token_secret='{your-access-token-secret}',
    )
```
